Overview

Welcome! This tab system uses only HTML and CSS — no JavaScript required. It works via hidden radio inputs and the :checked pseudo-class selector.

Details

Each tab is a <label> paired with a hidden <input type="radio">. When a radio is checked, CSS sibling selectors activate the matching panel.

Settings

To add more tabs, duplicate a radio input + label + panel triple, keeping the name attribute the same and giving each a unique id/for/id triplet.
